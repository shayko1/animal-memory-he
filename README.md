# משחק זיכרון: חיות (עברית)

משחק זיכרון מהיר ונגיש בעברית (RTL), עם חיות חמודות, תצוגה מקדימה קצרה, תמיכה במקלדת, ובדיקות אוטומטיות.

## פיצ'רים

- 🧠 משחק זיכרון עם 3 רמות קושי
- 👀 תצוגה מקדימה קצרה בתחילת משחק (מפחית תסכול)
- ⌨️ ניווט מקלדת (Tab/Enter)
- 🔊 אפקטי צליל עדינים (ניתן לכיבוי)
- 🧪 בדיקות יחידה (Vitest) + בדיקות E2E (Playwright)
- ✅ ESLint + Prettier + GitHub Actions

## פיתוח מקומי

```bash
npm install
npm run dev
```

## בדיקות

```bash
npm run test:run
npm run e2e
```

## Build

```bash
npm run build
npm run preview
```
